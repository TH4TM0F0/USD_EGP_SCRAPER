# USD to EGP Currency Scraper

## Overview
This project is a web scraper that retrieves the current exchange rate from USD to EGP (Egyptian Pounds) and provides functionality to track changes over time.

## Features
- Scrapes live exchange rates from multiple sources.
- Provides historical data and exchange rate trends.
- User-friendly interface to display the data.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/TH4TM0F0/USD_EGP_SCRAPER.git
   cd USD_EGP_SCRAPER
   ```
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
1. Run the scraper:
   ```bash
   python scraper.py
   ```
2. Access the fetched data in `data.json` or view it on the interface provided.
